ADVANCED DRIVER ASSISTANCE SYSTEM

  • Language: Python
  • Project date: 01 March, 2022
  • Framework: TensorFlow Lite
  • Network: SSD MobileNet-V2
  • Dataset: BDD100K, MS COCO, iBUG 300-W, Through Researchgate

What is our Avanced Driver Assistance System(ADAS) capable of??

My ADAS includes features like:
1. Object Detection
2. Driver Drowsiness
3. Blind Spot Detection
4. Lane Detection
5. Communication System

Getting to the Details

This project was an amalgamation of both software and hardware technologies.
SOFTWARE USED:
 1. Python IDE
2. Arduino IDE
HARDWARE USED:
 1. Raspberry Pi
2. Arduino Mega
3. Camera Module
4. Sensors: Heartbeat Sensors, Sharp Sensors
5. Lights and Buzzers

The system is modelled to detect the presence of potential hazards, monitor the surrounding environment, analyse the driver behaviour and send key information based on the data to take the necessary measures.
